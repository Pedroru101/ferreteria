Test - Resumen de Productos en Carrito

Este archivo prueba la implementación completa del resumen de productos en el dropdown del carrito flotante.

Funcionalidades a Probar:
  • Mostrar lista de productos con nombre, precio y cantidad
  • Mostrar subtotal por cada producto
  • Botones para aumentar/disminuir cantidad
  • Botón para eliminar productos
  • Mostrar total general del carrito
  • Botón "Generar Cotización" habilitado/deshabilitado
  • Mensaje cuando el carrito está vacío

Instrucciones de Prueba:

  2. Haz clic en el botón flotante del carrito (esquina inferior derecha)
  3. Verifica que se muestre el nombre, precio unitario y cantidad de cada producto
  4. Verifica que se muestre el subtotal de cada producto
  5. Prueba los botones - y + para cambiar cantidades
  6. Prueba el botón para eliminar productos
  7. Verifica que el total general se actualice correctamente
  8. Verifica que el botón "Generar Cotización" esté habilitado solo cuando hay productos
